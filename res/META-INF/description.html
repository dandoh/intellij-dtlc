Several dependently-typed experimental programming language support, in one plugin.

All your issues/pull requests will be replied very quickly.


Features


  • Mini-TT:
    • Running your code
    • Mini-TT executable management
    • Live templates
    • Syntax highlight, color settings page
    • Completion
    • Rename
    • Find usages
    • Goto declaration
    • Commenter, brace matcher
    • Create file, file icons
  • Agda:
    • Create file, file icons
    • Live templates context
  • OwO:
    • Agda compiler management
    • Create file, file icons
    • Live templates context
